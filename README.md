# evercount-privacy

Privacy policy for the [EverCount](https://github.com/Osirisborn89/Evercount) Android app, hosted via GitHub Pages.

**Live URL:** https://osirisborn89.github.io/evercount-privacy/

This URL is referenced inside the app (Settings → Privacy policy) and on the Google Play Store listing. It must always resolve.

## Editing

The policy is a single static `index.html`. Edit it, push to `main`, GitHub Pages re-deploys within ~1 minute.

## Why a separate repo

Keeping this out of the main `Evercount` repo means:
- The privacy URL stays stable even if the app source repo is ever renamed or made private.
- GitHub Pages is enabled on the public `main` branch without exposing the app source.
- Anyone — including Play Review bots — can read the policy without authentication.
